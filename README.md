# 📂 Smart File Organizer (Java)

A simple Java program that automatically organizes files in a folder based on file type.

## 🚀 Features

- Scans a folder
- Detects file types
- Creates folders automatically
- Moves files into categories

## 📁 Categories

- Images
- Documents
- Videos
- Audio
- Code
- Others
